# 배열의 깊은 복사, 얕은 복사의 방법
1. spread 연산자를 이용한 깊은복사
```javascript
    let obj1 = [1,2,3];
    let obj2 = {...obj1};
```

2. Object.assign()
```javascript
    let obj1 = [1,2,3];
    let obj2 = Object.assign({},obj1);
```

스프레드문법과 Object.assign 은 1차원 배열에서만 정상적으로 작동한다.
2차원으로 가면 다르게 해야된다
